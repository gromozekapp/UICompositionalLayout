# UICompositionalLayout

> UIKit `UICollectionViewCompositionalLayout` demo — Pinterest-style grid

[![Swift](https://img.shields.io/badge/Swift-UIKit-F05138?logo=swift&logoColor=white)]()
[![Status](https://img.shields.io/badge/type-portfolio%20exercise-blue)]()

UIKit collection view with compositional layout, built following a [YouTube tutorial](https://youtu.be/dmWFLRGWPUs).

---

## Features

- `UICollectionViewCompositionalLayout` with multiple section types
- Custom cell layouts
- UIKit + Swift

## Getting Started

```bash
git clone https://github.com/gromozekapp/UICompositionalLayout.git
cd UICompositionalLayout
open CompositionalLayout.xcodeproj
```

Select scheme **CompositionalLayout** → Run (`⌘R`).

## Requirements

- iOS 15+
- Xcode 15+
